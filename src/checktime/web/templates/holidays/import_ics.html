How to get an ICS file

You can get ICS calendar files from various sources:

  • From Google Calendar, go to Settings > Import & Export > Export
  • From Apple Calendar, select a calendar and use File > Export
  • From Microsoft Outlook, right-click on a calendar and select "Save calendar"
  • Download public holiday calendars from your country's official websites
